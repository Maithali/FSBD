The Details disclosure element

The details HTML element creates a disclosure widget
in which information is visible only when the widget is toggled into an "open" state.
A summary or label must be provided using the summary element.

Details Something small enough to escape casual notice.
System Requirements

Requires a computer running an operating system. The computer must have some memory and ideally some kind of long-term storage. An input device as well as some form of output device is recommended.
